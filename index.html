Markdeck

A lightweight Markdown and MDX viewer for Windows

Get it on Microsoft Store

Open anything

.md .mdx .markdown .mkd .mkdn .mdown .mdtxt .mdtext

Drag & drop

Drop files or folders directly into the viewer. No file picker needed.

File tree sidebar

Browse folders with a collapsible tree. Click any file to preview it instantly.

Dark & light themes

System, Light, Dark, and High Contrast themes. Matches your Windows preference.

Find in page

Search within any file with highlighted matches and navigation (Ctrl+F).

Print & Export PDF

Print any document or export it as a PDF directly from the viewer (Ctrl+P).

Configurable file types

Choose which file extensions to associate with Markdeck.

Auto-reload & recent files

Files reload when changed on disk. Recent files list keeps your work one click away.

Built for Windows

Native WinUI 3 app. Fast, small, no runtime dependencies.
