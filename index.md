# Markdeck

A lightweight Markdown and MDX viewer for Windows.

## Features

- Open `.md`, `.mdx`, `.markdown`, `.mkd`, `.mkdn`, `.mdown`, `.mdtxt`, `.mdtext` files
- Drag & drop files or folders
- File tree sidebar with folder browsing
- Dark and light theme support
- Configurable file type associations

## Download

Available on the Microsoft Store.
