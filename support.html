Getting Help

If you have questions, feedback, or need help with Markdeck:

  • GitHub Issues: Report a bug or request a feature
  • Email: Contact us through the Microsoft Store listing

Known Issues

  • MDX files are rendered as plain Markdown — JSX components are displayed as text, not executed
  • File type associations require Windows 10 version 1809 (17763) or later

System Requirements

  • Windows 10 version 1809 (build 17763) or later
  • x64 or ARM64 processor
